Classes Offered

Everything About Linux

This is a super fun, project based class about Linux. We'll set you up with your own Linux PC and start by showing how to install and configure the Operating System. From there we'll deploy some websites on the machine, we'll compile some video games from C++ code, we'll see how to set up a Netflix clone to stream your DVD library, and more. We'll also write some code in a few programming languages and look at a bunch of essential hacking techniques.

By the end of this class you'll be able to accomplish just about anything you want to do on a Linux computer. The class will amaze and inspire you. The instructor has over a decade of industry experience with Linux and will surely be able to answer any additional questions you have!

Price of the class includes your own Linux PC!

Embedded software development on Linux

This super fun, hands on class will involve controlling a variety of electro-mechanical devices from a raspberry pi or beaglebone computer. You'll learn all you need to build robots, design home IoT devices, and more.

Raspberry Pi and Beaglebone available for sale, or bring your own.
