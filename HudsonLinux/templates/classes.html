The Linux Workbook

$300 includes class + a PC

Learn everything you need to know to get started using Linux.

This is a super fun, project based class about Linux. We'll set you up with your own Linux PC and start by showing how to install and configure the Operating System. From there we'll deploy some websites on the machine, we'll compile some video games from C++ code, we'll see how to set up a Netflix clone to stream your DVD library, and more. We'll also write some code in a few programming languages and look at a bunch of essential hacking techniques.

By the end of this class you'll be able to accomplish just about anything you want to do on a Linux computer. The class will amaze and inspire you. The instructor has over a decade of industry experience with Linux and will surely be able to answer any additional questions you have!

Price of the class includes your own Linux PC! Small form factor PC with 512GB SSD plus minimum 4GB RAM. Class can run from one to three days depending on your convenience. If you live close you might prefer 3 small classes, but if you commute maybe one or two long days will be best for you.

How to Design Models and 3D Print

$250 includes 2 days of class + some 3D Prints

A great class to take before buying a 3d printer.

3D printers are cool - but there is more to the story. They are big, error prone, smelly, and could be bad for your health. Before you commit a corner of your small apartment to a huge stinky machine, you should come take this class.

Class outline:

  • At the beginning of the class you will look at a catalog of models, pick one you want, and we'll turn on the printer.
  • While the printer runs, we'll get started designing a model for you to print.
  • Then we will explore the variety of places on the internet you can look for models
  • Next, we learn how to use a slicer to turn an STL into GCODE.
  • We leave that model on the side while our first model finishes. printing
  • Now we move on to the resin printer and discuss how to use it.
  • Now we prep our STL for printing on the resin printer.
  • Then we start the resin printer.
  • Then we will clean and cure your resin print

At the end of the class you will have 2 or 3 prints depending on how much time we've had. Finally we will compare the 2 printing options. You will have smelled the fumes from both, seen the prep and clean up required for the two methods, and seen the quality of the output.

Price of the class includes all the material required: printers, filament, resin, gloves, cleaning supplies, etc..
